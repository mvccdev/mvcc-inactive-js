# mvcc-inactive-js

A small function used to check if a touchscreen or webpage has been inactive.

This is used to prevent screen burnout.

## Skill Prerequisites

A basic understanding of the following skills are necessary to use this project.

* [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)
* [ES6](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Tool Dependencies

Install the following tools to use this project in your local development environment.

* [Node.js](https://nodejs.org/)
* [Git](https://git-scm.com/)
* [GitHub Desktop](https://desktop.github.com/)

## Installing

Follow these steps to install this project.

	git clone https://github.com/mvcc-inactive-js/mvcc-inactive.git
	cd mvcc-inactive
	npm install
	npm build

## Building

Use the following CL commands to test or compile this project:

| Name          | Description                                          |
| ------------- | ---------------------------------------------------- |
| npm run build | Build the project                                    |

## Quick Example

**JS**

	mvcc-inactive(() => {
		//
		// Code to execute when the touchscreen or page is inactive.
		//
	}, 3000);
